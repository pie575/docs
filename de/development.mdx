<Info>
  **Voraussetzungen**:
  - Node.js Version 19 oder höher
  - Ein Docs-Repository mit einer `docs.json`-Datei
</Info>

Befolge diese Schritte, um Mintlify auf deinem Betriebssystem zu installieren und auszuführen.

<Steps>
<Step title="Mintlify-CLI installieren">

```bash
npm i -g mint
```
</Step>

<Step title="Lokal Vorschau anzeigen">

Navigiere zu deinem Docs-Verzeichnis, in dem sich deine `docs.json`-Datei befindet, und führe den folgenden Befehl aus:

```bash
mint dev
```

Eine lokale Vorschau deiner Dokumentation ist unter `http://localhost:3000` verfügbar.

</Step>
</Steps>

<div id="custom-ports">
  ## Benutzerdefinierte Ports
</div>

Standardmäßig verwendet Mintlify Port 3000. Sie können den Port, auf dem Mintlify ausgeführt wird, mit dem Flag `--port` anpassen. Um Mintlify beispielsweise auf Port 3333 zu starten, verwenden Sie diesen Befehl:

```bash
mint dev --port 3333
```

Wenn Sie versuchen, Mintlify auf einem bereits belegten Port zu starten, wird der nächste verfügbare Port verwendet:

```md
Port 3000 ist bereits belegt. Verwende stattdessen 3001.
```


<div id="mintlify-versions">
  ## Mintlify-Versionen
</div>

Bitte beachten Sie, dass jede CLI-Veröffentlichung einer bestimmten Mintlify-Version zugeordnet ist. Wenn Ihre lokale Vorschau nicht mit der Produktionsversion übereinstimmt, aktualisieren Sie bitte die CLI:

```bash
npm mint update
```


<div id="validating-links">
  ## Links validieren
</div>

Die CLI kann Sie beim Validieren von Links in Ihrer Dokumentation unterstützen. Um defekte Links zu finden, verwenden Sie den folgenden Befehl:

```bash
mint broken-links
```

<div id="troubleshooting">
  ## Fehlerbehebung
</div>

<AccordionGroup>
  <Accordion title='Fehler: Modul "sharp" konnte mit der darwin-arm64-Runtime nicht geladen werden'>

    Dies kann an einer veralteten Node-Version liegen. Probiere Folgendes:
    1. Entferne die aktuell installierte CLI-Version: `npm remove -g mint`
    2. Aktualisiere auf Node v19 oder höher.
    3. Installiere die CLI neu: `npm i -g mint`
  </Accordion>

  <Accordion title="Problem: Unbekannter Fehler">

    Lösung: Wechsle ins Home-Verzeichnis und lösche den Ordner `~/.mintlify`. Führe anschließend `mint dev` erneut aus.
  </Accordion>
</AccordionGroup>

Neugierig, was sich in der neuesten CLI-Version geändert hat? Sieh dir das [CLI-Changelog](https://www.npmjs.com/package/mintlify?activeTab=versions) an.
