<div id="authentication">
  ## Authentifizierung
</div>

Alle API-Endpunkte werden mithilfe von Bearer-Token authentifiziert und aus der Spezifikationsdatei übernommen.

```json
"security": [
  {
    "bearerAuth": []
  }
]
```
