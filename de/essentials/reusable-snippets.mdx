<div id="creating-a-custom-snippet">
  ## Einen eigenen Baustein erstellen
</div>

**Voraussetzung**: Legen Sie Ihre Baustein-Datei im Verzeichnis `snippets` an.

<Note>
  Jede Seite im Verzeichnis `snippets` wird als Baustein behandelt und nicht
  als eigenständige Seite gerendert. Wenn Sie aus dem Baustein eine eigenständige
  Seite machen möchten, importieren Sie den Baustein in eine andere Datei und
  binden Sie ihn als Komponente ein.
</Note>

<div id="default-export">
  ### Standardexport
</div>

1. Fügen Sie Ihrem Baustein Inhalte hinzu, die Sie an mehreren Stellen wiederverwenden möchten. Optional können Sie Variablen hinzufügen, die beim Import des Bausteins über Props belegt werden.

```mdx snippets/my-snippet.mdx
Hallo Welt! Das ist mein Inhalt, den ich seitenübergreifend wiederverwenden möchte. Mein Schlüsselwort des
Tages ist {word}.
```

<Warning>
  Der Inhalt, den du wiederverwenden möchtest, muss sich im Verzeichnis `snippets` befinden,
  damit der Import funktioniert.
</Warning>

2. Importiere den Baustein in deine Zieldatei.

```mdx destination-file.mdx
---
title: Mein Titel
description: Meine Beschreibung
---

import MySnippet from '/snippets/path/to/my-snippet.mdx';

## Überschrift

Lorem ipsum dolor sit amet.

<MySnippet word="bananas" />
```


<div id="reusable-variables">
  ### Wiederverwendbare Variablen
</div>

1. Exportieren Sie eine Variable aus Ihrer Baustein-Datei:

```mdx snippets/path/to/custom-variables.mdx
export const myName = 'mein Name';

export const myObject = { fruit: 'Erdbeeren' };
```

2. Importieren Sie den Baustein aus der Zieldatei und verwenden Sie die Variable:

```mdx destination-file.mdx
---
title: Mein Titel
description: Meine Beschreibung
---

import { myName, myObject } from '/snippets/path/to/custom-variables.mdx';

Hallo, mein Name ist {myName} und ich mag {myObject.fruit}.
```


<div id="reusable-components">
  ### Wiederverwendbare Komponenten
</div>

1. Erstellen Sie in Ihrer Baustein-Datei eine Komponente, die Props entgegennimmt, indem Sie
   sie als Pfeilfunktion exportieren.

```mdx snippets/custom-component.mdx
export const MyComponent = ({ title }) => (
  <div>
    <h1>{title}</h1>
    <p>... Baustein-Inhalt ...</p>
  </div>
);
```

<Warning>
  MDX wird im Rumpf einer Arrow Function nicht kompiliert. Verwende nach Möglichkeit HTML-Syntax oder einen Default-Export, wenn du MDX einsetzen musst.
</Warning>

2. Importiere den Baustein in deine Zieldatei und übergib die Props

```mdx destination-file.mdx
---
title: Mein Titel
description: Meine Beschreibung
---

import { MyComponent } from '/snippets/custom-component.mdx';

Lorem ipsum dolor sit amet.

<MyComponent title={'Benutzerdefinierter Titel'} />
```
