<div id="inline-code">
  ## Inline-Code
</div>

Um ein `Wort` oder eine `Wortgruppe` als Code zu kennzeichnen, setze sie in Backticks (`).

```
Um ein `Wort` oder eine `Phrase` als Code zu kennzeichnen, schließen Sie es in Backticks (`) ein.
```


<div id="code-blocks">
  ## Codeblöcke
</div>

Verwenden Sie [fenced code blocks](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks), indem Sie Code in drei Backticks einschließen, und geben Sie direkt nach den einleitenden Backticks die Programmiersprache Ihres Snippets an, um Syntaxhervorhebung zu erhalten. Optional können Sie nach der Programmiersprache auch den Namen Ihres Codes angeben.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hallo, Welt!");
    }
}
```

````md
```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hallo, Welt!");
    }
}
```
````
