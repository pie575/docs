<div id="authentication">
  ## 認証
</div>

すべての API（Application Programming Interface：アプリケーション・プログラミング・インターフェイス）エンドポイントは Bearer トークンで認証され、そのトークンは仕様ファイルから取得されます。

```json
"security": [
  {
    "bearerAuth": []
  }
]
```
