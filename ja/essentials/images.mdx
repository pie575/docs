<div id="image">
  ## 画像
</div>

<div id="using-markdown">
  ### Markdown を使う
</div>

[Markdown の構文](https://www.markdownguide.org/basic-syntax/#images)を使うと、次のコードで画像を追加できます

```md
![title](/path/image.jpg)
```

画像ファイルのサイズは5MB未満である必要があります。満たさない場合は、[Cloudinary](https://cloudinary.com/) や [S3](https://aws.amazon.com/s3/) などのサービスでホスティングすることをおすすめします。そのURLを使って埋め込めます。

<div id="using-embeds">
  ### 埋め込みの使用
</div>

画像をより柔軟にカスタマイズするには、[埋め込み](/ja/writing-content/embed) を使って画像を追加することもできます

```html
<img height="200" src="/path/image.jpg" />
```

<div id="iframes">
  ### iFrames
</div>

ドキュメント内に別の HTML ページを読み込みます。主に動画の埋め込みに使われます。

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
