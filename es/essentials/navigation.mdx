El menú de navegación es la lista de enlaces en cada sitio web.

Probablemente actualizarás `docs.json` cada vez que agregues una página nueva. Las páginas no se muestran automáticamente.

<div id="navigation-syntax">
  ## Sintaxis de navegación
</div>

Nuestra sintaxis de navegación es recursiva, lo que significa que puedes crear grupos de navegación anidados. No necesitas incluir `.mdx` en los nombres de las páginas.

<CodeGroup>

```json Regular Navigation
"navigation": {
  "tabs": [
    {
      "tab": "Docs",
      "groups": [
        {
          "group": "Getting Started",
          "pages": ["quickstart"]
        }
      ]
    }
  ]
}
```

```json Nested Navigation
"navigation": {
  "tabs": [
    {
      "tab": "Docs",
      "groups": [
        {
          "group": "Getting Started",
          "pages": [
            "quickstart",
            {
              "group": "Nested Reference Pages",
              "pages": ["nested-reference-page"]
            }
          ]
        }
      ]
    }
  ]
}
```

</CodeGroup>

<div id="folders">
  ## Carpetas
</div>

Simplemente coloca tus archivos MDX en carpetas y actualiza las rutas en `docs.json`.

Por ejemplo, para tener una página en `https://yoursite.com/your-folder/your-page`, crearías una carpeta llamada `your-folder` que contenga un archivo MDX llamado `your-page.mdx`.

<Warning>
  No puedes usar `api` como nombre de una carpeta a menos que la anides dentro de otra. Mintlify usa Next.js, que reserva la carpeta `api` en el nivel superior para llamadas internas del servidor. Un nombre de carpeta como `api-reference` sí se acepta.
</Warning>

```json Navigation With Folder
"navigation": {
  "tabs": [
    {
      "tab": "Docs",
      "groups": [
        {
          "group": "Nombre del Grupo",
          "pages": ["your-folder/your-page"]
        }
      ]
    }
  ]
}
```


<div id="hidden-pages">
  ## Páginas ocultas
</div>

Los archivos MDX que no estén incluidos en `docs.json` no aparecerán en la barra lateral, pero podrán accederse a ellos mediante la búsqueda o enlazando directamente.
