<div id="inline-code">
  ## Código en línea
</div>

Para indicar que una `palabra` o `frase` es código, enciérrala entre acentos graves (`).

```
Para marcar una `palabra` o `frase` como código, enciérrala entre comillas invertidas (`).
```


<div id="code-blocks">
  ## Bloques de código
</div>

Usa [bloques de código delimitados](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) encerrando el código entre tres acentos graves y, a continuación, indica el lenguaje de programación de tu fragmento reutilizable para obtener el resaltado de sintaxis. Opcionalmente, también puedes escribir el nombre de tu fragmento después del lenguaje de programación.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("¡Hola, Mundo!");
    }
}
```
