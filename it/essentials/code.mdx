<div id="code-blocks">
  ## Blocchi di codice
</div>

Usa i [fenced code blocks](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) racchiudendo il codice tra tre backtick e, subito dopo i backtick iniziali, indica il linguaggio di programmazione del tuo snippet per ottenere lâ€™evidenziazione della sintassi. In alternativa, puoi anche aggiungere un nome al tuo codice dopo il linguaggio di programmazione.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Ciao, mondo!");
    }
}
```
