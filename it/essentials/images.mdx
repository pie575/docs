  ## Immagine
</div>

<div id="using-markdown">
  ### Utilizzo di Markdown
</div>

La [sintassi Markdown](https://www.markdownguide.org/basic-syntax/#images) consente di aggiungere immagini usando il seguente codice

```md
![title](/path/image.jpg)
```

Tieni presente che la dimensione del file immagine deve essere inferiore a 5 MB. In caso contrario, ti consigliamo di ospitarla su un servizio come [Cloudinary](https://cloudinary.com/) o [S3](https://aws.amazon.com/s3/). Puoi quindi utilizzare quell’URL per l’incorporamento.

<div id="using-embeds">
  ### Utilizzo degli embed
</div>

Per avere più flessibilità con le immagini, puoi anche usare gli [embed](/it/writing-content/embed) per aggiungere immagini

```html
<img height="200" src="/path/image.jpg" />
```

<Tip>
  Mintlify supporta i [tag HTML nel Markdown](https://www.markdownguide.org/basic-syntax/#html). Questo è utile se preferisci i tag HTML alla sintassi Markdown e ti consente di creare documentazione con flessibilità pressoché illimitata.
</Tip>

<div id="iframes">
  ### iFrames
</div>

Carica un’altra pagina HTML all’interno del documento. Usato principalmente per incorporare video.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
