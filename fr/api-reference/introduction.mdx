<div id="authentication">
  ## Authentification
</div>

Tous les endpoints de l’API sont authentifiés à l’aide de jetons Bearer et récupérés à partir du fichier de spécification.

```json
"security": [
  {
    "bearerAuth": []
  }
]
```
