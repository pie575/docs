<div id="creating-a-custom-snippet">
  ## 创建自定义片段
</div>

**前置条件**：必须在 `snippets` 目录中创建片段文件。

<Note>
  `snippets` 目录中的任何页面都会被视为片段，不会渲染为独立页面。
  如果你想基于该片段创建独立页面，请将其导入到另一个文件中，
  并作为组件进行调用。
</Note>

<div id="default-export">
  ### 默认导出
</div>

1. 将你想在多个位置复用的内容添加到你的片段文件中。你也可以添加变量，在导入该片段时通过 props 传入以填充。

```mdx snippets/my-snippet.mdx
你好世界！这是我想要在多个页面中重复使用的内容。我今天的关键词是 {word}。
day is {word}.
```

<Warning>
  要实现导入，需确保要复用的内容位于 `snippets` 目录中。
</Warning>

2. 将片段导入目标文件。

```mdx destination-file.mdx
---
title: 我的标题
description: 我的描述
---

import MySnippet from '/snippets/path/to/my-snippet.mdx';

## 标题

Lorem ipsum dolor sit amet.

<MySnippet word="bananas" />
```


<div id="reusable-variables">
  ### 可复用变量
</div>

1. 从你的片段文件中导出变量：

```mdx snippets/path/to/custom-variables.mdx
export const myName = '我的名字';

export const myObject = { fruit: '草莓' };
```

2. 从目标文件导入片段，并使用该变量：

```mdx destination-file.mdx
---
title: 我的标题
description: 我的描述
---

import { myName, myObject } from '/snippets/path/to/custom-variables.mdx';

你好，我的名字是 {myName}，我喜欢 {myObject.fruit}。
```


<div id="reusable-components">
  ### 可复用组件
</div>

1. 在片段文件中，使用箭头函数定义并导出一个组件，使其通过 props 接收参数。

```mdx snippets/custom-component.mdx
export const MyComponent = ({ title }) => (
  <div>
    <h1>{title}</h1>
    <p>... 片段内容 ...</p>
  </div>
);
```

<Warning>
  MDX 无法在箭头函数的函数体内编译。能用 HTML 语法时尽量使用；如确需使用 MDX，请改用默认导出。
</Warning>

2. 将该片段导入目标文件，并传入 props

```mdx destination-file.mdx
---
title: 我的标题
description: 我的描述
---

import { MyComponent } from '/snippets/custom-component.mdx';

Lorem ipsum dolor sit amet.

<MyComponent title={'自定义标题'} />
```
