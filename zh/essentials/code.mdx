<div id="inline-code">
  ## 行内代码
</div>

要将某个 `word` 或 `phrase` 标示为代码，请用反引号（`）将其括起来。

```
要将`单词`或`短语`标记为代码，请用反引号（`）将其包围。
```


<div id="code-blocks">
  ## 代码块
</div>

使用[围栏代码块](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks)：将代码放在三个反引号中，并在起始反引号后注明该片段所使用的编程语言，以启用语法高亮。你也可以选择在编程语言后补充代码名称。

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("你好，世界！");
    }
}
```

````md
```java HelloWorld.java

    }
````
