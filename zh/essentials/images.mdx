  ## 图像
</div>

<div id="using-markdown">
  ### 使用 Markdown
</div>

通过 [Markdown 语法](https://www.markdownguide.org/basic-syntax/#images)，你可以使用以下代码插入图片

```md
![title](/path/image.jpg)
```

请注意，图片文件大小必须小于 5MB。否则，建议将其上传到 [Cloudinary](https://cloudinary.com/) 或 [S3](https://aws.amazon.com/s3/) 等服务进行托管。然后使用该 URL 进行嵌入。

<div id="using-embeds">
  ### 使用嵌入
</div>

为获得更灵活的图片自定义方式，你也可以使用[嵌入](/zh/writing-content/embed)来添加图片

```html
<img height="200" src="/path/image.jpg" />
```

<div id="iframes">
  ### iFrames
</div>

在文档中嵌入并加载另一个 HTML 页面。最常用于嵌入视频。

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
