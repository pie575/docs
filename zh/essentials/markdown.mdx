<div id="text-formatting">
  ## 文本格式
</div>

我们支持大多数 Markdown 格式。只需在文本两侧加上 `**`、`_` 或 `~` 即可完成格式化。

| 样式         | 写法               | 效果              |
| ------------- | ------------------ | ----------------- |
| 粗体          | `**bold**`         | **bold**          |
| 斜体          | `_italic_`         | _italic_          |
| 删除线        | `~strikethrough~`  | ~strikethrough~   |

这些格式可以组合使用。例如，输入 `**_bold and italic_**` 可得到 **_bold and italic_** 文本。

上标和下标需要使用 HTML，即在文本两侧添加 `<sup>` 或 `<sub>`。

| 文本样式     | 写法                       | 效果                      |
| ------------- | -------------------------- | ------------------------- |
| 上标          | `<sup>superscript</sup>`   | <sup>superscript</sup>    |
| 下标          | `<sub>subscript</sub>`     | <sub>subscript</sub>      |

<div id="linking-to-pages">
  ## 链接到页面
</div>

你可以通过将文本包裹在 `[]()` 中来添加链接。例如，写作 `[link to google](https://google.com)` 会生成指向 [link to google](https://google.com) 的链接。

文档内页面的链接应使用以根路径为基准的相对路径。也就是说，应包含完整的文件夹路径。例如，`[link to text](/writing-content/text)` 会链接到我们组件部分中的“Text”页面。

像 `[link to text](../text)` 这样的相对路径链接会打开得更慢，因为我们无法同样高效地对其进行优化。

<div id="singleline">
  ### 单行
</div>

要创建引用块，请在段落前添加一个 `>`。

> 多萝西跟着她穿过了她城堡里许多美丽的房间。

```md
> Dorothy 跟着她走过了城堡里许多美丽的房间。
```


<div id="multiline">
  ### 多行
</div>

> 多萝西跟着她穿过城堡里许多华美的房间。
>
> 女巫吩咐她把锅壶刷洗干净、把地打扫干净，并不停地往火里添柴。

```md
> Dorothy 跟着她穿过城堡中许多美丽的房间。
>
> 女巫让她清洗锅碗瓢盆，打扫地板，并用木柴保持火焰燃烧。
```
