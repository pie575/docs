<div id="authentication">
  ## 身份验证
</div>

所有 API（Application Programming Interface，应用程序编程接口）端点均使用 Bearer 令牌进行身份验证，相关配置将从规范文件中读取。

```json
"security": [
  {
    "bearerAuth": []
  }
]
```
