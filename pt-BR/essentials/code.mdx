<div id="code-blocks">
  ## Blocos de código
</div>

Use [blocos de código cercados](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) envolvendo o código em três crases e, em seguida, coloque a linguagem de programação do seu snippet logo após as crases iniciais para obter realce de sintaxe. Opcionalmente, você também pode indicar o nome do seu arquivo após a linguagem de programação.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Olá, Mundo!");
    }
}
```
