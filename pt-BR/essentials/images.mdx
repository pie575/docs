  ## Imagem
</div>

<div id="using-markdown">
  ### Usando Markdown
</div>

A [sintaxe do Markdown](https://www.markdownguide.org/basic-syntax/#images) permite adicionar imagens usando o seguinte código

```md
![title](/path/image.jpg)
```

Observe que o tamanho do arquivo da imagem deve ser inferior a 5 MB. Caso contrário, recomendamos hospedar o arquivo em um serviço como o [Cloudinary](https://cloudinary.com/) ou o [S3](https://aws.amazon.com/s3/). Em seguida, você pode usar essa URL e incorporar a imagem.

<div id="using-embeds">
  ### Usando embeds
</div>

Para ter mais flexibilidade com imagens, você também pode usar [embeds](/pt-BR/writing-content/embed) para adicioná-las

```html
<img height="200" src="/path/image.jpg" />
```

<div id="iframes">
  ### iFrames
</div>

Carrega outra página HTML dentro do documento. Geralmente usado para incorporar vídeos.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
